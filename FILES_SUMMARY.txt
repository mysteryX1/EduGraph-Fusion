================================================================================
新增的 API 端点 (8 个)
================================================================================

RAG 模块 (3 个):
  1. POST /api/rag/index
     └─ 建立向量索引，支持 metadata 和 processed 数据源

  2. POST /api/rag/query
     └─ 查询知识库，返回 answer + citations + source_chunks

  3. GET /api/rag/status
     └─ 获取索引状态 (indexed, chunk_count, textbook_count)

反馈模块 (2 个):
  4. POST /api/feedback
     └─ 处理自然语言命令 (保留/删除/拆分/合并)

  5. GET /api/feedback/summary
     └─ 获取反馈统计摘要

报告模块 (3 个):
  6. POST /api/report/generate
     └─ 生成整合报告 (report/整合报告.md)

  7. GET /api/report/latest
     └─ 获取最新报告内容

  8. GET /api/report/summary
     └─ 获取报告统计摘要
