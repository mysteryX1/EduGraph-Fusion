# EduGraph Fusion 需求分析

## 项目概述

EduGraph Fusion 是一个教材知识底座系统，旨在通过自动化的教材文件处理、知识图谱构建、智能合并和检索等功能，为教育工作者和学生提供高效的知识管理和学习辅助工具。

## 核心需求

### 1. 文件上传与解析
- **需求**：支持多种格式教材上传（PDF、Markdown、纯文本）
- **功能**：自动提取文件内容、识别章节结构、提取关键信息
- **约束**：单个文件最大 100MB，支持异步解析

### 2. 知识图谱构建
- **需求**：自动从教材内容提取知识概念和关系
- **功能**：建立概念间的关联关系，支持多教材融合
- **展示**：可视化知识图谱，支持交互式浏览

### 3. 图谱合并与去重
- **需求**：合并多个教材的知识图谱，消除重复和冗余
- **功能**：自动检测相似概念，提供合并建议
- **交互**：支持用户确认和决策

### 4. 知识检索（RAG）
- **需求**：基于向量数据库的语义检索
- **功能**：支持自然语言提问，返回相关内容和源引用
- **优化**：支持参数调整，如返回结果数量

### 5. 反馈收集
- **需求**：收集用户对系统的反馈
- **功能**：支持多种反馈类型（缺陷、建议、问题等）
- **分析**：反馈汇总和趋势分析

### 6. 综合报告生成
- **需求**：自动生成系统分析报告
- **功能**：包括统计信息、质量评分、建议等
- **格式**：支持多种导出格式

## 用户场景

### 教育工作者
1. 上传课程教材
2. 自动提取知识点
3. 建立知识图谱
4. 生成教学辅助资料

### 学生
1. 浏览知识概念
2. 语义搜索学习内容
3. 理解概念间的关系
4. 获得学习建议

### 系统管理员
1. 管理教材库
2. 监控系统性能
3. 查看反馈信息
4. 生成工作报告

## 功能要求

### 前端功能
1. **单页应用布局**
   - 左侧：教材管理区
   - 中间：知识图谱可视化
   - 右侧：Tab 功能面板

2. **交互功能**
   - 文件拖拽上传
   - 图谱节点点击交互
   - 参数实时调整
   - 实时 loading 反馈

3. **可视化展示**
   - ECharts 图表
   - 节点颜色区分来源
   - 节点大小表示频率
   - 支持缩放和拖拽

4. **错误处理**
   - 详细的错误提示
   - Mock 数据 fallback
   - 网络异常恢复

### 后端功能
1. **API 接口**
   - 文件上传和处理
   - 教材信息管理
   - 知识图谱操作
   - RAG 索引和查询
   - 反馈收集和分析
   - 报告生成

2. **数据处理**
   - 多格式文件解析
   - 异步后台处理
   - 向量化和索引
   - 缓存优化
